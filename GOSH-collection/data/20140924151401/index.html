Créer, éditer ou supprimer des pages

À quoi servent les pages ?

Les pages sont idéales pour publier du contenu que vous souhaitez partager avec vos lecteurs. Par exemple : À propos, Infos personnelles, CV, Liens favoris, etc.

Comment créer une page ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Cliquez sur "Pages".
  5. Cliquez sur "Ajouter une page".
  6. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, si vous désirez ajouter une page Informations personnelles, vous pouvez écrire simplement /infospersos ou /infos.
  7. Ajoutez un titre dans le champ “Titre de la page”.
  8. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  9. Cliquez sur “Enregistrer”. 
Ou :
  1. Cliquez sur “Personnaliser” à droite du Tableau de bord. Assurez-vous d'avoir sélectionné le blog que vous souhaitez éditer dans le menu des blogs.
  2. Cliquez sur “Ajouter une page”.
  3. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, si vous désirez ajouter une page Informations personnelles, vous pouvez écrire simplement /infospersos ou /infos.
  4. Ajoutez un titre dans le champ “Titre de la page”.
  5. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  6. Cliquez sur “Enregistrer”. 

Comment éditer une page ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Cliquez sur la page que vous désirez éditer.
Ou :
  1. Cliquez sur “Personnaliser” à droite du Tableau de bord. Assurez-vous d'avoir sélectionné le blog que vous souhaitez éditer dans le menu des blogs.
  2. Cliquez sur la page que vous désirez éditer.

Comment supprimer une page ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Cliquez sur l'icône "X" située à côté de la page que vous désirez supprimer.
Ou :
  1. Cliquez sur “Personnaliser” à droite du Tableau de bord. Assurez-vous d'avoir sélectionné le blog que vous souhaitez éditer dans le menu des blogs.
  2. Cliquez sur l'icône "X" située à côté de la page que vous voulez supprimer.

Affichage standard vs affichage personnalisé

Comment personnaliser l'apparence d'une page ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Cliquez sur “Ajouter une page” et choisissez l'option “Affichage personnalisé” dans le menu déroulant.
  5. Insérez le code personnalisé dans l'éditeur HTML.
  6. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, si vous désirez ajouter une page Informations personnelles, vous pouvez écrire simplement /infospersos ou /infos.
  7. Ajoutez un titre dans le champ “Titre de la page”.
  8. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  9. Cliquez sur “Enregistrer”.
Ou :
  1. Cliquez sur “Personnaliser” à droite du Tableau de bord. Assurez-vous d'avoir sélectionné le blog que vous souhaitez éditer dans le menu des blogs.
  2. Cliquez sur “Ajouter une page” et choisissez l'option “Affichage personnalisé” dans le menu déroulant.
  3. Insérez le code personnalisé dans l'éditeur HTML.
  4. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, si vous désirez ajouter une page Informations personnelles, vous pouvez écrire simplement /infospersos ou /infos.
  5. Ajoutez un titre dans le champ “Titre de la page”.
  6. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  7. Cliquez sur “Enregistrer”.

Quelle est la différence entre l'affichage standard, l'affichage personnalisé et la redirection ?

  • L'affichage standard : reproduit le thème de votre blog. La page ressemble donc au reste de votre blog.
  • L'affichage personnalisé : utilise un code personnalisé pour concevoir l'apparence d'une page.
  • Redirection : ﻿renvoie un visiteur vers une autre adresse web. Par exemple, vous pouvez rediriger vos visiteurs vers les billets étiquetés avec un tag précis ou vers des sites externes tels que votre compte Twitter.

Pages avec redirection

Comment créer une page qui redirige vers mes billets étiquetés avec un même tag ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Choisissez l'option “Pages” en bas à gauche de la page.
  5. Cliquez sur “Ajouter une page”.
  6. Choisissez “Redirection” dans le menu déroulant.
  7. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, /montag, mais remplacez "montag" par le tag que vous voulez utiliser.
  8. Dans le champ “Rediriger sur”, ajouter /tagged/montag à la fin de l'adresse web de votre blog. N'oubliez pas de remplacer “montag” par le tag que vous souhaitez utiliser.
  9. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  10. Cliquez sur “Enregistrer”.
Ou :
  1. Cliquez sur “Personnaliser” à droite du Tableau de bord. Assurez-vous d'avoir sélectionné le blog que vous souhaitez éditer dans le menu des blogs.
  2. Choisissez l'option “Pages” en bas à gauche de la page.
  3. Cliquez sur “Ajouter une page”.
  4. Choisissez “Redirection” dans le menu déroulant.
  5. Entrez une adresse web pour la page après la barre oblique (/). Par exemple, /montag, mais remplacez "montag" par le tag que vous voulez utiliser.
  6. Dans le champ “Rediriger sur”, ajouter /tagged/montag à la fin de l'adresse web de votre blog. N'oubliez pas de remplacer “montag” par le tag que vous souhaitez utiliser.
  7. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  8. Cliquez sur “Enregistrer”.

Comment créer une page servant à rediriger vers un autre site web ?

  1. Cliquez sur "Paramètres" (l'icône en forme d'engrenage) en haut de votre tableau de bord.
  2. Cliquez sur le blog que vous désirez éditer, à gauche de la page.
  3. Cliquez sur "Personnaliser" dans la section Thèmes.
  4. Choisissez l'option “Pages” en bas à gauche de la page.
  5. Cliquez sur “Ajouter une page”.
  6. Choisissez “Redirection” dans le menu déroulant.
  7. Entrez une adresse web pour la page après la barre oblique (/). Si, par exemple, vous voulez créer une page de redirection vers votre compte Twitter, vous pouvez écrire /twitter et si vous voulez rediriger vers votre compte Facebook, vous pouvez écrire /facebook.
  8. Dans le champ “Rediriger sur”, écrivez l'adresse web vers laquelle vous souhaitez rediriger vos visiteurs. Si vous voulez créer une page de redirection vers votre compte Twitter, l'adresse web de votre page sera www.twitter.com/nomd'utilisateur (votre nom d'utilisateur est votre @jesuis).
  9. Cliquez sur le bouton bascule "Afficher un lien vers cette page".
  10. Cliquez sur “Enregistrer”.

  3. Cliquez sur “Ajouter une page”.

Résolution de problèmes

Pourquoi mon blog n'affiche-t-il pas les liens des pages que je crée ?

Tous les thèmes recommandés sont conçus pour afficher un lien vers les pages que vous créez sur votre blog. Si malgré tout ils n'apparaissent pas, une des raisons ci-dessous peut expliquer le problème :

  • Vous utilisez peut-être un thème qui ne prend pas en charge cette fonctionnalité.
  • Vous utilisez du code HTML personnalisé qui empêche votre blog de recevoir la plus récente version de votre thème.

Comment m'assurer que les liens vers mes pages s'affichent sur mon blog ?

  • Changez de thème ou installez la plus récente version du thème que vous utilisez.
  • Créez un lien HTML pointant vers votre page dans le champ "Description" ou à l'intérieur du code HTML de votre blog. Par exemple :
    <a href="/pageurl">Ma Page !</a>
    (remplacez “pageurl” par l’adresse web de votre page)
  • Éditez le code HTML de votre thème. Cliquez sur "Personnaliser" et insérez le code manuellement. Pour plus de détails à ce sujet, consultez la documentation sur les thèmes.
